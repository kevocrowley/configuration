# opencode-theme

This repository contains custom theme configuration files for OpenCode.

## Shiro Telin Theme

The `shirotelin.json` file defines a custom theme for the OpenCode terminal UI.

**Important:** This theme does NOT include a light theme variant. All `light` color values are identical to their `dark` counterparts, resulting in a single theme appearance regardless of system light/dark mode settings.

This is by design - the theme is intended for users who prefer a specific color palette regardless of ambient lighting conditions.
